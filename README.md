# Causal Inference: The Effect of Temperature on Walmart Sales

## 📌 Project Overview
**Objective:** To determine the causal relationship between localized temperature fluctuations and retail sales volume at Walmart. By isolating external variables, this project aims to provide concrete, data-driven insights into how weather impacts consumer purchasing behavior.

## 📊 Data Description
The analysis utilizes retail sales data combined with historical weather records. 
* **Target Variable:** Weekly sales volume.
* **Key Independent Variables:** Temperature, regional identifiers, and temporal markers. 
* **Data Structure:** Cleaned and formatted into structured data frames for econometric modeling.

## 🛠️ Methodology & Tools
This project was conducted using **R** and documented via **Quarto** for reproducible reporting. The statistical approach focuses on isolating causality rather than mere correlation:
* **Fixed Effects Models:** Deployed to control for unobserved, time-invariant variables across different regions and store locations, ensuring the isolated effect of temperature is accurate.
* **Polynomial Regression:** Implemented polynomial temperature variables to account for non-linear relationships (e.g., the difference in consumer behavior during extreme cold versus extreme heat).

## 💡 Key Findings
* **Temperature does causally effect sales but this effect is not linear. Sales are highest moderate temperatures and lowest at extreme temperatures.
* **As temperature increases, we would expect sales to decrease in the warmer months and increase in the colder months. Over the course of a year,
    as temperatures increase, these results should even out and there should be little to no cumulative effect on sales from temperature.
